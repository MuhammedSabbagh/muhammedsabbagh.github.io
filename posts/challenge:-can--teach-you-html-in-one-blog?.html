HTML, Which stands for (HyperText Markup Language), is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript(JS). Wikipedia

<p></p>

What Each part of this means?

Let’s start with each one from top to bottom..

<!DOCTYPE html>

this line lets the browser know that it’s opening an HTML file and thus rendering it as a webpage, it’s needed in any webpage.

<head>
</head>

this requires a bit of explanation… as we discussed before, using HTML without CSS will leave us with nothing but bad styled text webpage and thus we need css and javascript, but how do we include them? this is where <head></head> comes in handy! it allows us to link css and js files to our html and thus giving it Style and functionality. this topic is for another blog but let’s keep going with the other commands

<body></body>

this one contain all the visible elements in the webpage wheter hyperlinks,texts,articles..

<p></p>

this command let us write text to our webpage otherwise how can we write blogs :D.

<title></title>

this one give the website a title, when your page loads check this..

<a href="https://www.google.com/">Google</a>

this one works similar to <p></p> except this one leads to a webpage when clicked.
